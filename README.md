## Project Name
Water Level Indicator

## Project Scope
This project is designed to monitor the water level in a tank and indicate different levels using LEDs and a buzzer. It helps prevent water overflow and enables efficient water management.

## Components Used
- Arduino Uno
- Water Level Sensor
- LEDs
- Buzzer
- Jumper Wires

## Working Principle
The sensor detects the water level and sends data to the Arduino. Based on the level, LEDs glow to indicate low, medium, and high water levels. When the tank is full, the buzzer turns on.

## Output
The system successfully indicates the water level and alerts the user when the tank is full.
